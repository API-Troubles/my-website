QR Code Generator

Need a shortened URL with analytics? Check out url.felixgao.dev!

Tip: Start your URL with https:// so devices will recognize it as a link!

Tip: Set a higher level (Quartile/High) if you want to print this outside, where the QR code will take damage from the outdoors. Set a lower level if you need the QR code to be small for some reason (Low). Keep it normal otherwise (Medium) :).

QR Codes aren't stored/processed/sold to Google, this website is completely static! :yay:
